# Screenshots Folder

Place your extension screenshots here:
- `demo.gif` - Extension demonstration animation
- `widget-ui.png` - Extension popup / settings interface
- `widget-custom.png` - Floating widget interface example

## Taking Screenshots

1. Load the extension in Chrome
2. Take screenshots at 1920x1080 or similar resolution
3. Crop to show relevant UI elements
4. Save with descriptive names
